
Filiere	: Informatique
Niveau	: 4

				Travail pratique sur l'UE intitule VISION PAR ORDINATEUR		

Le but de ce TP est d"effectuer des traitements d'images.
Plusieurs opérations ont été effectuées:
	1-  OPERATIONS ARITHMETIQUES 
    2-  OPERATIONS LOGIQUES
    3-  INVERSE DE L'IMAGE
    4-  BINARISATION
    5-  HISTOGRAMME
    6-  INTERPOLATION
    7-  TRANSFORMATIONS LINEAIRES
    8-  CONVOLUTION
    9- DETECTION DE CONTOURS
    10- SEGMENTATION
    11- CALCUL DU CONTRASTE
    12- TRANSFORMEE DE HOUGH
    13- ROTATION

Pour l'executer, positionnez-vous dans le dossier, puis tapez la commande make pour effectuer la compilation et l'execution(consulter le Makefile pour plus d'indications).

Des images sont disponibles dans le dossier Images. Une fois traitées, l'image est stockée dans le dossier Images/NouvellesImages.
